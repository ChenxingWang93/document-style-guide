n = 3.

C(u) = a_{0} + a_{1}u + a_{2}u^{2}, 0<=u<=1,

The cubic, 三次

C(u) = a_{0} + a_{1}u + a_{2}u^{2} + a_{3}u^{3},

is a very general curve; 非常一般曲线

it can be truly ***twisted*** three- dimensional curve, not lying in a single plane 可以是真正的扭曲三维曲线，而不是位于一个平面上 (Figure1.8a);

it can have an inflection point 它可以有一个拐点 (Figure1.8b);

a cusp 一个尖峰 (Figure1.8c);

or a Loop 或一个循环 (Figure 1.8d).

A twisted curve results if 一个扭曲弧线的结果 如果

a_{0},a_{1},a_{2},a_{3}

do not lie in a unique plane. 不位于一个平面上

an inflection point on a planar curve 平面曲线上的一个拐点 is defined as a point where the curve is smooth 该点处的曲线是平滑的 (no cusp).

and the tangent line at that point passes through the curve. 该点处的 切线穿过曲线

This implies a change in the turning direction of a curve. 意味着曲线的转向方向发生变化

At an inflection point, 在拐点处

either

C''(u) = 0,

or

C'(u) || C''(u).

A necessary(but not sufficient) condition for a cusp at 对于尖点的必要不充分条件

u = u_{0}

is

C'(u) = 0

(velocity zero) 速度为 0

Conditions for a loop to occur are also known. 循环发生的条件 可见 (see [Ferg66, 67, 69, 93; Forr70, 80; Wang81; Ston89; Su89]).
